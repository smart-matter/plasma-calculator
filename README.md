# plasma-calculator
Python-based benchmark  predictions for Plasma cash implementations

## Input parameters

1. Tree Depth
2. Number of account
3. Trades Per Second (Per User)
4. Number of Pair (Stub, defined to 1)
5. Block producing Speed Per Minute

## Output parameters

1. TPS
2. Exit speed
3. Support costs
4. Number of constraints per zkSNARK
5. CRS length
